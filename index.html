01. Install

Install high_voltage in your local system

$ gem install high_voltage

Include high_voltage in your gemfile

gem "high_voltage"

For Rails versions prior to 3.0, use the 0.9.2 tag of high_voltage

$ gem "high_voltage", "~> 0.9.2"
